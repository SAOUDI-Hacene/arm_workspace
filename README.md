# 6-DOF Robotic Arm Simulation with ROS2 Jazzy

A professional 6-DOF robotic arm simulation using ROS2 Jazzy, Gazebo Harmonic, and RViz2.

## 🎯 Features

- ✅ **6 Degrees of Freedom (6-DOF)** - Full articulated arm motion
- ✅ **Realistic Arm Design** - Professional industrial-grade geometry
- ✅ **Logical Joint Limits** - Safe and constrained motion
- ✅ **ROS2 Jazzy** - Latest ROS2 distribution
- ✅ **RViz2 Visualization** - Real-time 3D visualization
- ✅ **Joint State Publisher GUI** - Easy interactive control

## 📋 System Requirements

- OS: Ubuntu 24.04 LTS
- ROS2: Jazzy
- GPU: Recommended

## 🚀 Quick Start

### Launch UR5 Robot
```bash
source /opt/ros/jazzy/setup.bash
ros2 launch ur_description view_ur.launch.py ur_type:=ur5
```
